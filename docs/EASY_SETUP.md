# 🎯 초간단 설정 가이드

## 1️⃣ Supabase 접속
- 테스트: https://supabase.com/dashboard/project/roeurruguzxipevppnko
- 로그인하세요

## 2️⃣ SQL Editor 열기
- 왼쪽 메뉴에서 `SQL Editor` 클릭
- `New query` 버튼 클릭

## 3️⃣ 복사 & 붙여넣기
- `database/simple_setup.sql` 파일 열기
- **전체 복사** (Ctrl+A, Ctrl+C)
- SQL Editor에 **붙여넣기** (Ctrl+V)

## 4️⃣ 실행
- `Run` 버튼 클릭 (또는 Ctrl+Enter)
- "Success" 나오면 완료! ✅

## 5️⃣ 테스트
1. 브라우저에서 로그인
2. 새 일기 작성
3. 로그아웃 후 다른 계정으로 로그인
4. 이전 일기가 안 보이면 성공! 🎉
